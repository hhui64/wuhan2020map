# 新型冠状病毒发热门诊查询工具

[![Build Status](https://travis-ci.com/hhui64/wuhan2020map.svg?branch=master)](https://travis-ci.com/hhui64/wuhan2020map)
![GitHub](https://img.shields.io/github/license/hhui64/wuhan2020map)

覆盖全国 300+ 城市，快速查询所属城市中设立的发热门诊机构以及定点医疗机构。

## 数据来源

本工具的数据来源于`国家人口健康科学数据中心`以及`各地卫健委官网`，确保信息来源真实可靠；数据源可能存在更新不及时、某些字段遗失以及缺少部分城市数据或医院信息的情况，如发现遗漏、错误等问题请提直接提交 [Issue](https://github.com/hhui64/wuhan2020map/issues/new)。
